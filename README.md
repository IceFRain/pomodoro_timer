# 🚀 核心技能（续命指南）

## 🍅 1. 番茄时钟 · 劳逸结合
* ⏱️ **自定时间**：自由配置疯狂专注的工作时长（默认 25 min）与回血续航的休息时长（默认 5 min）。
* 🎮 **便携控制**：悬浮窗与托盘图标实时显示进度。通过右键菜单一键「开始/暂停/继续」，打工节奏由你掌控！

## 💧 2. 饮水记录 · 吨吨吨吨
* 🎯 **目标设定**：量化你的每日饮水目标（比如 1800 ml），还能设定你专属发财杯的单杯容量（比如 500 ml）。
* 🥤 **便捷打卡**：悬浮窗/托盘支持“记一杯”快速快乐续杯！直观的进度条百分比同步更新，看着它涨满超有成就感。

## ⚙️ 3. 贴心配置 · 绝不加戏
* 🚀 **开机自启**：一键配置，开机即刻自动开启健康打工的元气一天！
* 🎛️ **悬浮窗自由**：主窗口提供独立勾选框，番茄钟、饮水记录的悬浮窗想开哪个开哪个，只留下你需要的，断舍离极简风。

### 📥 第一步：抓取软件
点击下方传送门，去发布页面把最新版的生存物资带回家：

👉 **[下载链接/GitHub](https://github.com/IceFRain/cubicle-survivor/releases)**

👉 **[下载链接/Gitee](https://gitee.com/icefrain/cubicle-survivor/releases)**

> **💡 电脑版本怎么选？**
> * 🚀 比较新的高配电脑 / Windows 11 以上：无脑选 **`x64`** 版本。
> * 👵 饱经风霜的生产力老电脑：请尊选 **`x86`** 版本。
> * 🔼 升级：选择对应的 **`upgrade`** 轻量替换

### 🏃 第二步：即刻起飞
1. 解压下载好的物资压缩包，双击里面的那个小可爱： **`cubicle_survivor`**。
2. 软件默认会乖巧地以悬浮窗姿态诞生在你的屏幕正中间。
3. **强烈建议**：双击它打开主界面，把 **「开机自启」** 勾上！下次上班开机，它就会默默陪着你一起升级打怪啦！
