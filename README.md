# 🍅 格子间幸存者
专注，本该无形。 这是一个为你桌面减负的超轻量番茄工作法工具。拒绝臃肿的主界面，用最克制的悬浮窗，陪你度过高效的每一天。

# ✨ 为什么选择它？
大部分番茄钟要么太大挡住屏幕，要么藏在系统托盘，想看一眼时间还得去点图标。
这款番茄钟专为“重度屏幕工作者”设计，有这些特点：

## 🧩 极致轻量
启动即化身为一个精美的小型悬浮窗与托盘图标，始终置顶显示，可自由拖动到屏幕上空闲的位置，任务栏亦可，进度一目了然。

## 🖱️ 极简操作
- 双击（悬浮窗/托盘） ➔ 唤醒主设置，定制你的专注节奏。
- 右键（悬浮窗/托盘） ➔ 掌控全局（开始、暂停、继续、退出），无需打开大窗口。

## 🔔 沉浸体验
默认经典 25分钟工作 + 5分钟休息 循环。结束时通过 Windows 原生通知轻声提醒，绝不打扰你的下一秒思考。

# 🚀 开始使用
👉[下载地址](https://github.com/IceFRain/pomodoro_timer/releases)

下载对应压缩包(目前仅支持windows)，解压后双击“pomodoro_timer.exe”运行，悬浮窗会在屏幕中央显示，拖动到理想的位置，右键悬浮窗，开始即可。

# 🎯 经典番茄工作法推荐
如果你不知道怎么设置最适合自己，可以试试软件内置的默认黄金法则：
工作 25 分钟：保持绝对专注，只做一件事。
休息 5 分钟：眼睛离开屏幕，站起来喝杯水。
